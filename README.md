# Franks Angels

A nonprofit platform helping youth baseball fields thrive in underserved communities.

**Mission:** Connect communities with youth baseball fields that need support, ratings, and funding to improve facilities for young athletes.

Live at: https://franks-angels.vercel.app

## What It Does

Franks Angels empowers communities to support youth baseball by:

**For Communities:**
- Discover youth baseball fields in your area
- View field conditions, ratings, and photos
- See which fields need the most help
- Donate directly to improve facilities

**For Field Supporters:**
- Rate fields on: grass, dirt, dugouts, fences, bleachers
- Leave reviews about conditions
- Identify fields needing urgent attention
- Track improvement over time

**For Field Managers:**
- Add your youth baseball field to the platform
- Showcase current conditions and needs
- Receive community support and donations
- Track ratings and feedback

## Features

- ✅ **Field Directory** — Search and browse youth baseball fields
- ✅ **Field Ratings** — Rate grass, dirt, dugouts, fences, bleachers
- ✅ **Field Health Scores** — See which fields need help most
- ✅ **Add New Fields** — Submit fields to the database
- ✅ **Donation Support** — Contribute to field improvements
- ✅ **Photo Gallery** — View field conditions visually
- ✅ **Community Reviews** — Read and write field reviews

## Tech Stack

- **Frontend:** Next.js 14 (App Router), React, TypeScript
- **Styling:** TailwindCSS with custom navy theme
- **Backend:** Supabase (Auth, PostgreSQL Database, Storage)
- **Icons:** Lucide React
- **Deployment:** Vercel
